# Wandering South

Wandering South is a responsive tourism website project designed to showcase beautiful travel destinations, tour packages, and travel experiences across South India.
The project is built using **HTML, CSS, and JavaScript** with a clean and modern user interface.

---

##  Features

*  Explore famous tourist destinations
*  Hills, beaches, and occasion-based travel sections
*  Fully responsive design for mobile, tablet, and desktop
*  Search functionality for tourist places
*  Travel package page
*  About page with company information
*  Contact and enquiry section
*  Modern and clean UI design

---

##  Technologies Used

* HTML5
* CSS3
* JavaScript
